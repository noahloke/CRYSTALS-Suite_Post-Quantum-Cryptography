CRYSTALS Kyber

Kyber is an IND-CCA2-secure key encapsulation mechanism designed to be secure against attacks from both classical and quantum computers. This is paired with a public key encryption system, consisting of a public and secret key pair. It would typically encapsulate AES-256 bit keys for secure network communications. Its status as IND-CCA2 means that it has indistinguishability under an adaptive chosen ciphertext attack, meaning that its ciphertext does not reveal any useful information about the original plaintext (Jiang, 2018).

Kyber has 3 different base parameter sets: Kyber-512, Kyber-768, and Kyber-1024. These implementations seek to match the security levels of AES-128, AES-192, and AES-256, respectively. There is also an additional variant for these parameter sets, Kyber-90s, which is implemented when hardware support for the symmetric primitives is available. It uses AES-256 in counter mode and SHA2 instead of SHAKE. This adds the parameter sets of Kyber-512-90s, Kyber-768-90s, and Kyber-1024-90s.
