Welcome

This website is a student research project on the CRYSTALS (Cryptographic Suite for Algebraic Lattices) post-quantum cryptographic package. Both of its cryptographic primitives, Kyber, a key-encapsulation mechanism, and Dilithium, a digital signature algorithm, were chosen by NIST (National Institute of Standards and Technology) for PQC Standardization in 2022. These algorithms are designed to withstand cryptanalytic attacks by current and future quantum computers, as well as classical computers. They both rely on the hardness of mathematical problems over module lattices.

Methodology: Lattice Based Cryptography

A lattice is a grid-like structure in n dimensions, generated with points created from integer combinations of a set of vectors and their opposites.

The common 2D coordinate plane with points at integer coordinates can also be thought of as a lattice formed from the set of vectors (0, 1) and (1, 0). Each point in this lattice can be reached using a repetitive combination from the given set of vectors and their respective opposites. The set of vectors that generated the lattice are the secret set. A public set could be (3, 1) and (1, 2) (with their opposites as a given: (-3, -1), (-1, -2)). This set can still can reach every point on the lattice but it will be more difficult.

Blue: (0, -1) + (0, -1) + (0, -1) + (1, 0) = (1, -3)

Red: (3, 1) + (-1, -2) + (3, 1) + (-1, -2) + (-1, -2) + (-3, -1) + (1, 2) = (1, -3)


Reliance on the Hardness of the Mathematical Problems:

Shortest Vector Problem (SVP): shortest possible combination of public set of vectors from the origin to target point

Closest Vector Problem (CVP): find the lattice point closest to the target point, must generate lattice with public set and compare all surrounding points

Kyber - Learning With Errors (LWE): must solve a set of linear equations (vectors) with small random noise (dimensional offset) added

Dilithium - Module Learning With Errors (MLWE): uses modules instead of vectors like in LWE. While a lattice is formed over integers, a module lattice is formed over a ring of integers.


NIST PQC Standarization 2022:

Both CRYSTALS algorithms (Lattice Based) were prioritized for their security and performance

FALCON (Lattice Based) – smaller signature than Dilithium

SPHINCS+ (Hash Based) – implementation to not fully rely on lattices
